16 Registers:
0 - 12 = General
13: Frame Pointer
14: Stack Pointer
15: Program Counter

Flags: NZCV
Flag codes: 0 / 1 / X


Load Store:
    LDR Rd Rm <offset>
    STR Rd Rn <offset>

Branch:
    JMP RX <offset>
    JIF Flags RA <offset>

DP: 
    OP R1 = R2 op R3 
    OP R1 = op R2

    OPIF <Flagcode> ... = conditional execution
    OPS = affect flags
